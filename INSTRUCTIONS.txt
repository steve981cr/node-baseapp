Install packages:
npm install
Add some more packages:
npm i sequelize sqlite3 express-validator

Install sequelize-cli globally:
npm install -g sequelize-cli

Set up the Sequelize file structure
node_modules/.bin/sequelize init
sequelize init

Generate a model:
sequelize model:generate --name Article --attributes title:string,content:string,published:boolean

Change the config/config.json file for sqlite.

Run migrations:
If sequelize-cli was installed locally:
node_modules/.bin/sequelize db:migrate
globally: sequelize db:migrate

Check out the sqlite3 database schema from the CLI:
sqlite3 database.sqlite3
.schema
Exit the REPL: Ctrl+D

Generate a seeder script:
node_modules/.bin/sequelize seed:generate --name seed-article
sequelize seed:generate --name seed-article

Seed the db:
If sequelize-cli was installed locally:
node_modules/.bin/sequelize db:seed:all
globally: sequelize db:seed:all

Confirm they are in the db:
sqlite3 database.sqlite3
SELECT * FROM Articles;
Ctrl+D
